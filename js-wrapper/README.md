# @thiagovt-dev/xodex-cli

Wrapper em Node para chamar o **Xodex** (core em Python).

## Instalação

### NPM (registro público)
```bash
npm i -g @thiagovt-dev/xodex-cli
```

> Requer o Xodex (Python) disponível no PATH:

```bash
pipx install xodex
# ou
python -m pip install --user xodex
```

## Uso

```bash
xodex --help
```
